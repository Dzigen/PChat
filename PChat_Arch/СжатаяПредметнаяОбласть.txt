 Система предоставляет способ взаимодействия пользователей для обмена сообщениями. К сообщениям относятся: текстовые сообщения,фотографии, видео, аудиозаписи и файлы.

Информация о пользователях будет храниться в общем списке, к которому можно будет обращаться для поиска людей и их добавления в свой личный список контактов. Пользователь может удалить контакт из личного списка. К информации о пользователе относится: имя, фотография.

Предусмотрены 3 способа передачи сообщений между пользователями: личная переписка, канал и беседа.

В личной переписке состоят только два пользователя, которые имеют к данному чату доступ и могут обмениваться сообщениями. Пользователь может удалить свою переписку с определенным контактом. Для личной переписки у пользователя есть опция выбрать соединение для обмена сообщениями с другим контактом без посредничества со стороны вспомогательного удалённого устройства.

Беседа предполагает обмен сообщениями между несколькими участниками одновременно, при этом их чат не виден другим пользователям. Беседа создаётся одним пользователем, который становится администратором беседы и получает права: добавлять/удалять участников, передавать статус администратора, изменять описание, название и фотографию беседы, добавлять чат-ботов и делегировать права, описанные ранее, другим участникам беседы. Только у администратора есть возможность делегировать права, добавить чат-ботов и передать статус администратора. Участники могут выйти из беседы. Если администратор выходит из беседы, то его статус переходит к первому приглашённому участнику.

Канал позволяет просматривать своё содержимое всем без права на редактирование. Пользователи могут подписываться на канал и оставлять комментарии под опубликованными сообщениями. Канал создаётся одним участником, который становиться администратором и получает права: публиковать/удалять сообщения в канале, передавать свой статус администратора, удалить канал, отвечать на комментарии пользователей от лица канала, блокировать определённым или всем пользователям возможность комментирования публикаций, изменять описание, название и фотографию канала, добавлять чат-ботов , а также делегировать права, описанные ранее другим пользователям. Только у администратора есть возможность удалить канал, передать статус администратора, добавить чат-ботов и делегировать права.

В канале и беседе есть врзможность создания опросов для своих подписчиков/участников. Право создания опроса есть только у администратора канала/беседы, которое он может делегировать.

В личной переписке и беседе доступна опция видео-аудио связи.

Каждый пользователь может выбрать тему оформления своей системы обмена сообщениями из предложенного списка. Пользователю будет доступна тёмная и светлая темы, а также несколько других нейтральных полутонов.

Пользователь может добавлять/удалять дополнительные компоненты приложения, описанные далее.

Может быть добавлен компонент прослушивания музыки. Компонент включает в себя поисковик, плеер и список сохранённой музыки. Пользователь может добавить, найденную через поисковую систему, музыку в свой список. Возможно удаление музыки из списка.

Может быть добавлен компонент для получения уведомлений и передачи сообщений из других социальных сетей и электронной почты. Пользователь может добавить несколько мессенджеров из доступного списка для отслеживания новых сообщений. Пользователь может удалить мессенджер из отслеживаемого списка.

Может быть добавлен компонент для передачи больших файлов между парой пользователей, которые нельзя отправить через личную переписку, беседу или канал
