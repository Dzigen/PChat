Функциональные требования
1. Регистрация и вход ( H )

    Прохождение этапа регистрации в мессенджере и создание личного профиля; (ФТ1 п1)
    Авторизация при входе в мессенджер посредством ввода логина и пароля.(ФТ1 п2)

2. Информация профиля ( H )

    Просмотр и изменение персональной информации аккаунта: имени, фотографии пользователя;(ФТ2 п1)
    Просмотр информации другого пользователя.(ФТ2 п2)

3 Общая база пользователей/каналов ( H )

    Поиск пользователей и каналов; (ФТ3 п1)
    Добавление пользователей в "личный список контактов"; (ФТ3 п2)
    Добавление каналов в "список чатов каналов". (ФТ3 п3)

4.Список личных контактов ( H )

    Поиск контактов; (ФТ4 п1)
    Создание чата с конкретным контактом и добавление в "cписок чатов личных переписок"; (ФТ4 п2)
    Удаление конкретных контактов. (ФТ4 п3)

5.Список чатов личных переписок ( H )

    Поиск контактов; (ФТ5 п1)
    Удаление личных переписок; (ФТ5 п2)

6.Список чатов бесед( H )

    Поиск бесед; (ФТ6 п1)
    Удаление истории бессед; (ФТ6 п2)

7.Список чатов каналов( H )

    Поиск каналов; (ФТ7 п1)
    Удаление канала из списка; (ФТ7 п2)

8. Личная переписка ( H )

    Просмотр содержания личной переписки с определённым контактом; (ФТ8 п1)
    Обмен сообщениями; (ФТ8 п2)
    Изменение конкретных сообщений; (ФТ8 п4)
    Удаление конкретных сообщений; (ФТ8 п5)
    Выбор pear2pear соединения для обмена сообщениями.
    По завершении переписки через pear2pear соединение, выбор: сохранить переписку на локальных устройствах, удалить переписку или сохранить её на сервере. (ФТ8 п6)

9. Беседы ( M )

    Создание беседы; (ФТ79 п1)
    Просмотр содержания беседы; (ФТ9 п2)
    Обмен сообщениями между участниками; (ФТ9 п3)
    Функционал администратора беседы: (ФТ9 п4)
        Изменение информации беседы: описание, название и фотографию; (ФТ9 п4.1)
        Добавление/удаление участников; (ФТ9 п4.2)
        Создание опросов; (ФТ9 п4.3)
        Делегирование прав, описанных ранее, другим участникам беседы; (ФТ9 п4.4)
        Передача статуса администратора; (ФТ9 п4.5)
        Добавление/удаление чат-ботов; (ФТ9 п4.6)
            Чат-ботов для подключения можно выбрать из списка доступных или указать локальное расположение собственных. (ФТ9 п4.6.1)
        Удаление беседы;(ФТ9 п4.7)
    Возможность участника покинуть беседу. (ФТ9 п5)

10. Каналы ( M )

    Создание канала; (ФТ10 п1)
    Подписка на канал: добавление в "список чатов каналов";
    Просмотр содержания канала; (ФТ10 п2)
    Комментирование публикаций в канале; (ФТ10 п3)
    Функционал администратора Канала: (ФТ10 п4)
        Добавить/удалить публикации в канале; (ФТ10 п4.1)
        Ответить на комментарии пользователей от лица канала; (ФТ10 п4.2)
        Блокировать определённым или всем пользователям возможность комментирования публикаций; (ФТ10 п4.3)
        Изменить описание, название и фотографию канала; (ФТ10 п4.4)
        Делегировать права, описанные ранее другим пользователям; (ФТ10 п4.5)
        Удалить канал; (ФТ10 п4.6)
        Передать статуса администратора; (ФТ10 п4.7)
        Добавление/удаление чат-ботов: (ФТ10 п4.8)
            Чат-ботов для подключения можно выбрать из списка доступных или указать локальное расположение собственных. (ФТ10 п4.8.1)

11. Удаление аккаунта пользователя( M )
12.Интерфейс прросмотра фото, видео, GIF и прослушивания аудиозаписей в чатах( M )

    Возможность загрузить и отправить файл в чате личной переписки, канала и беседы; (ФТ12 п1)
    Возможность просмотра фото, видео, GIF и прослушивания аудиозаписей внутри чата. (ФТ12 п2)

13. Аудио и видео звонок ( L )

    Возможность совершать аудио-видео звонок в личной переписке и беседе; (ФТ13 п1)
    Возможность принять или отклонить звонки от других контактов; (ФТ13 п2)
    Возможность начать видео-аудио звонки в беседе; (ФТ13 п3)
    Возможность включить/выключить микрофон, включить/выключить камеру на своём устройстве, запустить/завершить демонстрацию экрана и прекратить соединение; (ФТ13 п4)
    Возможность включить/выключить камеру на своём устройстве; (ФТ13 п5)
    Возможность запустить/завершить демонстрацию экрана; (ФТ13 п6)
    Участники аудио-видео звонков могут прекратить соединение; (ФТ13 п7)
    Для личной переписки пользователь, получивший уведомление о звонке, может принять или отклонить его. Звонок прекращается, когда один из его участников прекращает соединение; (ФТ13 п8)
    Для беседы участник может начать конференцию, после чего другие участники беседы смогут к ней подключиться. Другие пользователи не могут подключиться к конференции; Конференция прекращается, когда отключается последний участник. (ФТ13 п9)

14. Буферизация данных ( H )

    Временного хранение данных чатов пользователей на стороне пользователя; (ФТ14 п1)
    Возможность пользователя сохранить данные своих чатов в локальном хранилище. (ФТ14 п2)

15. Дополнительные компоненты ( M )

    Возможность пользователю добавлять/удалять дополнительные компоненты приложения (прослушивания музыки, создания уникальной темы оформления интерфейса своего приложения, получение уведомлений и отправки сообщений из других социальных сетей и электронной почты, пересылка больших файлов, которые нельзя отправить через личную переписку, беседу или канал, между парой пользователей). (ФТ15 п1)

16. Кастомизация интерфейса( M )

    Выбор встроенной темы интерфейса. Пользователю будет доступна тёмная и светлая темы, а также несколько других нейтральных полутонов; (ФТ16 п1)
    Возможность загрузить собственные фотографии, которые будут использоваться в качестве заднего фона для интерфейсных элементов. (ФТ16 п2)

17. Пересылки больших файлов ( L )

    Cоздание pear2pear соединения для передачи файлов конкретному пользователю из "списка личных контактов". (ФТ17 п1)

18. Мониторинг социальных сетей ( L )

    Добавление мессенджеров из доступного списка в "список отслеживаемых мессенджеров": Вконтакте, Telegram; (ФТ18 п1)
    Просмотр содержания отслеживаемых месенджеров через встроенный интерфейс; (ФТ18 п2)
    Удаление мессенджеров из "списка отслеживаемых мессенджеров". (ФТ18 п3)

19. Прослушивание музыки ( L )

    Прослушивание музыки через встроенный плеер; (ФТ19 п1)
    Поиск музыки через встроенную поисковую систему; (ФТ19 п2)
    Добавление музыки в "список сохранённой музыки"; (ФТ19 п3)
    Удаление музыки из "списка сохранённой музыки"; (ФТ19 п4)
    Обращение к локальной папке на устройстве для поиска и прослушивания музыки; (ФТ19 п5)
    Для чатов добавляется опция: прикрепить к сообщению аудиозапись из " списка сохранённых аудиозаписей". (ФТ19 п6)
