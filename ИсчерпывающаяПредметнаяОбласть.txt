Приложение представляет собой систему межсетевого взаимодействия пользователей для обмена сообщениями. К сообщениям относятся: текстовые сообщения,фотографии, видео, аудиозаписи и файлы.

Информация о пользователях будет храниться в общей базе, к которой можно будет обращаться для поиска людей и их добавления в свой личный список контактов. Пользователь может удалить контакт из личного списка.

В приложении предусмотрены 3 способа передачи сообщений между пользователями: личная переписка, канал и беседа.

В личной переписке состоят только два пользователя, сообщения которых не видны никому, кроме них. Пользователь может удалить свою переписку с определенным контактом. Сообщения на второй стороне также будут удалены.
Для личной переписки у пользователя есть опция выбрать pear2pear соединение для обмена сообщениями с другим контактом без посредничества со стороны сервера. По завершении переписки через pear2pear соединение, пользователи смогут выбрать: сохранить переписку на локальных устройствах, удалить переписку или сохранить её на сервере.

Беседа предполагает обмен сообщениями между большим количеством участников одновременно, но их сообщения также являются скрытыми от остальных пользователей приложения. Беседа создаётся одним пользователем, который становится администратором беседы и получает права: добавлять/удалять участников, передавать статус администратора, изменять описание, название и фотографию беседы, добавлять чат-ботов и делегировать права, описанные ранее, другим участникам беседы. Чат-ботов для подключения можно выбрать из списка доступных или указать локальное расположение собственных. Только у администратора есть возможность делегировать права, добавить чат-ботов и передать статус администратора. Участники могут выйти из беседы. Если администратор выходит из беседы, то его статус переходит к первому приглашённому участнику.

Канал позволяет просматривать своё содержимое всем без права на редактирование.  Пользователи могут только оставлять комментарии под опубликованными сообщениями в канале. Информация о каналах будет храниться в общей базе, к которой можно обращаться для поиска и добавления в свой личный список отслеживаемых каналов. Канал создаётся одним участником, который становиться администратором и получает права: публиковать/удалять сообщения в канале, передавать свой статус администратора, удалить канал, отвечать на комментарии пользователей от лица канала, блокировать определённым или всем пользователям возможность комментирования публикаций, изменять описание, название и фотографию канала, добавлять чат-ботов , а также делегировать права, описанные ранее другим пользователям. Чат-ботов для подключения можно выбрать из списка доступных или указать локальное расположение собственных. Только у администратора есть возможность удалить канал, передать статус администратора, добавить чат-ботов и делегировать права.

Для личной переписки, беседы и канала предусмотрен интерфейс для просмотра фото, видео, gif-файлов и прослушивания аудиозаписей.
В канале и беседе есть возможность создания опросов для своих участников/читателей с выбором одного или нескольких вариантов ответа из перечисленного набора. Право создания опроса есть только у администратора беседы/канала, которое он может делегировать.

В личной переписке и беседе доступна опция видео-аудио связи. Пользователи смогут: общаться через микрофоны, включить/выключить камеру на своём устройстве, запустить/завершить демонстрацию экрана и прекратить соединение. Для личной переписки пользователь, получивший уведомление о звонке, может принять или отклонить его. Звонок прекращается, когда один из его участников прекращает соединение. Для беседы участник может начать конференцию, после чего другие участники беседы смогут к ней подключиться. Другие пользователи не могут подключиться к конференции. Конференция прекращается, когда отключается последний участник.

Каждый пользователь может выбрать тему оформления своего приложения из предложенного списка. Пользователю будет доступна тёмная и светлая темы, а также несколько других нейтральных полутонов. Также будет доступна возможность сгрузить данные из локального хранилища. Пользователь сможет удалить файлы, картинки, видео, аудио-файлы полученные и загруженные из сообщений личных переписок, каналов и бесед.

Пользователь может добавлять/удалять дополнительные компоненты приложения, описанные далее.

Может быть добавлен компонент прослушивания музыки. Компонент включает в себя поисковик,плеер и список сохранённой музыки. Пользователь может добавить, найденную через поисковую систему,  музыку в свой список. Возможно удаление музыки из списка. Пользователь также может локально обратиться к папке на своём компьютере для поиска и прослушивания аудиозаписей. Для личной переписки, канала и беседы появляется новая опция: прикрепить к сообщению через приложение аудиозапись из сохранённого списка или локальной папки.

Может быть добавлен компонент создания уникальной темы оформления интерфейса своего приложения. Пользователь может загрузить собственные фотографии, которые будут использоваться в качестве заднего фона для интерфейсных элементов.

Может быть добавлен компонент для получения уведомлений и отправки сообщений из других социальных сетей и электронной почты. Пользователь может добавить несколько мессенджеров из доступного списка для отслеживания новых сообщений. Для них в приложении предусмотрен интерфейс для прямого взаимодействия с урезанным функционалом. Пользователь может удалить мессенджер из отслеживаемого списка.

Может быть добавлен компонент для пересылки больших файлов, которые нельзя отправить через личную переписку, беседу или канал,  между парой пользователей. Пользователь сможет выбрать файл для отправки и пользователя из своего списка контактов в качестве пункта назначения. Будет создано pear2pear соединение, через которое будет передаваться файл.
